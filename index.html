machelper for OS X

machelper is a command line tool for OS X to help mac/OSx86 users to do some specific tasks. Originally the idea was to ease my pain as I don't like so many apps installed on my PC for performing specific and very simple tasks. But after the creation of the tool, I realised that it could help others if I shared it with others.

Features

There are lots of features of this simple command line tool, and still adding:

  • Creation of bootable USB (USB installer) for both native mac and OSx86 users.
    • with Clover (for native Mac, use -excludeClover option)
    • only supports EFI/UEFI
    • ability to configure ethernet manually (with -ethernet option)
    • uses config.plist for laptops directly from RehabMan's repo (with -config option)
    • applies MBR patch (with -mbrPatch option)
    • replace kernel (with -replaceKernel option)
  • Kernel Extension (kext) administration
    • backup & install/uninstall a kext
    • fix permission(s)
    • rebuild cache
  • Extract DSDT/SSDTs as well as disassemble them
  • Mount EFI partition (needs further improvement)
  • Other tools:
    • copy Clover from USB installer to HDD/SSD (GPT only)
    • backup & uninstall (delete) kext from Single User Mode
    • list unsigned kext
    • allow the very installation of machelper on USB installer for further use
    • replace kernel from Single User Mode
    • disable Hibernation
    • export kernel log (not yet released)
    • backup & delete/restore all the graphics kext from /S/L/E (not yet released) & many more...
